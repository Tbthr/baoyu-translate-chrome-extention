# 翻译指令

## 翻译风格
**Storytelling（叙事风格）**：以引人入胜的方式讲述，如同一位优秀的中文散文作者在向读者娓娓道来自己的真实经历。保持原文的自嘲幽默感和智识底色，让中文读者感受到一个有趣的灵魂在讲述他的真实故事。

## 内容背景
这是一篇发表在 Substack 上的个人随笔，作者是英国自由撰稿人 James O'Malley。文章探讨现代都市生活中的社交孤立问题，以及如何通过主动构建社群来重建人际关系。文章融合了个人叙事、社会观察和政治哲学思考。

## 翻译注意事项

### 核心原则
1. **重写而非翻译**：用自然流畅的中文重写，仿佛一位优秀的中文母语散文作者在讲述自己的故事。
2. **准确性优先**：事实、论点、逻辑必须与原文完全一致。
3. **自然流畅**：使用地道中文语序，将英语长句拆分为简短自然的中文句子。

### 风格要点
- 保留作者的英式自嘲和冷幽默——幽默感是本文的灵魂
- 叙事节奏：个人故事部分轻松亲切，理论分析部分保持思辨但不学究
- 脚注中的插科打诨要保留，语气可以更随意
- 斜体字在中文中对应使用加粗或适当保留标记
- 直接引语（英文带引号的）保留引号

### 特殊处理
- 保留所有 markdown 格式：标题、加粗、斜体、图片、链接、代码块、脚注
- 人名和专有名词：首次出现时用"中文（英文原名）"格式，后续直接用中文
- 网络用语和流行文化引用：找到中文语境中的等价表达，或在括号中说明
- 英文中的双关语和文字游戏：尽量保留，如无法保留则在脚注中说明
